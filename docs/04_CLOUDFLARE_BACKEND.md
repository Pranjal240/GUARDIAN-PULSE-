## CLOUDFLARE WORKERS KYA HAI?

Simple explanation: Ye tumhara server hai jo cloud mein hai.

- Tumhara JavaScript code Cloudflare globally run karta hai
- FREE: 100,000 requests/day
- Fast: Milliseconds mein respond karta hai
- Koi server maintain nahi karna — bas code likho aur deploy karo

> ✅ **Backend code Antigravity ne already generate kar diya hai** in `backend/src/`
> Ab bas deploy karna hai!

---

## DEPLOY KARO (Terminal se — Guardian Pulse folder mein)

```bash
# Step 1: Backend folder mein jao
cd backend

# Step 2: Dependencies install karo
npm install

# Step 3: Cloudflare login (browser open hoga)
wrangler login

# Step 4: Build + deploy
wrangler deploy
# Output: https://guardian-pulse-api.YOUR-NAME.workers.dev ✅
```

## CUSTOM DOMAIN ADD KARO

1. Cloudflare Dashboard → Workers & Pages → guardian-pulse-api
2. Settings → Triggers → Custom Domains → "Add Custom Domain"
3. Type: `api.guardianpulse.in`
4. Done! Ab tumhara API `https://api.guardianpulse.in` pe hai ✅

## ALERT ESCALATION LOGIC

```
t = 0 sec    → FCM push to user app (120-second countdown starts)
t = 2 min    → If NOT resolved: SMS + Email to emergency contacts
t = 8 min    → If still NOT resolved: SMS to contact 2 + critical email
```

Escalation `scheduled()` cron automatically chalti hai har minute.

## R2 MEDIA BUCKET (Chat photo/video uploads)

1. Cloudflare Dashboard → R2 Object Storage → "Create bucket"
2. Name: `guardian-pulse-media`
3. Public access enable karo → copy the public URL (e.g. `https://pub-xxxxx.r2.dev`)
4. `wrangler secret put R2_PUBLIC_URL` → paste that URL
